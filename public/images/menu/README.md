# Directorio de imágenes del menú

Este directorio contiene las imágenes organizadas por categorías del menú:

## Estructura:
- `entrantes/` - Imágenes de los platos de entrantes
- `ensaladas/` - Imágenes de ensaladas
- `bocadillos/` - Imágenes de bocadillos
- `burgers/` - Imágenes de hamburguesas
- `carnes/` - Imágenes de platos de carne
- `postres/` - Imágenes de postres
- `bebidas/` - Imágenes de bebidas

## Formato recomendado:
- Formato: JPG o PNG
- Resolución: mínimo 800x600px
- Aspecto: 4:3 o 16:9
- Peso: máximo 500KB por imagen

## Placeholder:
Se usa `placeholder-food.jpg` como imagen por defecto cuando no existe la imagen específica del plato.
