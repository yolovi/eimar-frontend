# Placeholder para imágenes del restaurante

Este archivo README.md sirve como placeholder para las imágenes reales del restaurante.

## Estructura sugerida:

```
public/images/restaurant/
├── ambiente1.jpg        # Imagen del comedor/ambiente interior
├── ambiente2.jpg        # Imagen de la terraza
├── comida1.jpg         # Plato destacado 1
├── comida2.jpg         # Plato destacado 2
├── local-exterior.jpg  # Fachada del restaurante
└── local-cocina.jpg    # Imagen de la cocina (si disponible)
```

## Para añadir imágenes reales:

1. Reemplaza estos archivos placeholder por imágenes reales
2. Mantén los nombres de archivo para que funcione automáticamente
3. Optimiza las imágenes para web (máximo 1200px de ancho recomendado)
4. Usa formatos modernos como .webp si es posible

## Dimensiones recomendadas:
- Aspecto ratio: 16:9 o 4:3
- Resolución: 1200x675px (16:9) o 1200x900px (4:3)
- Tamaño de archivo: < 500KB por imagen
